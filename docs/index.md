我是苏易文，一名从材料科学全力转向计算机科学的浙大学生。促使我做出这一决定的，是对"计算"本身力量的着迷——尤其是在一次实践中，我了解到字节跳动的工程师如何通过优化算法将网约车相关犯罪率降低20%，这让我深刻意识到代码背后承载的巨大社会价值。
